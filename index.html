Q8. What's the difference between window vs document?

Ans:

Document:

1. It represents the document loaded inside the window or browser.

2. The properties related to it are stored in the document object

3. Example: document.title will return the title of the document

logically:

        document:{ properties}

Window:

1. It represents the browser window in which you are seeing the content.

2. The properties related to it are stored in the window object.

3. Example: window.document.title will return the title of the document.

logically:
        window:{
            document:{properties}
        }
